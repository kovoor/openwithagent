# Install OpenWithAgent

OpenWithAgent is a static site. Clone it, serve locally, and optionally deploy to Cloudflare Pages.

## Steps

1. Clone: `git clone https://github.com/kovoor/openwithagent.git`
2. Serve locally: `cd openwithagent/public && python3 -m http.server 8080`
3. Open `http://localhost:8080` in your browser
4. To deploy: connect the repo to Cloudflare Pages with build output directory set to `public/`
