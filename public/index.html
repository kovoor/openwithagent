Paste the badge markdown into your README. Users see a clean "Open with AI" button.

3

Users click and go

They pick their AI tool (Claude Code, Cursor, Codex) and get a ready-to-paste command.

# Install MyTool

Clone the repo, install dependencies, and run the setup script.

## Steps
1. Clone: `git clone https://github.com/you/mytool.git`
2. Install: `cd mytool && npm install`
3. Setup: `npm run setup`
4. Verify: `npm test`
